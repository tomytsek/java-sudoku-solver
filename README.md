# java-sudoku-solver
Java Sudoku Solver  A Sudoku game and solving application developed in Java using object-oriented programming principles.  The project provides puzzle generation, validation and solving functionality through an interactive graphical user interface.

------Features------
Interactive Sudoku board 
Puzzle validation 
Sudoku solving algorithms 
User-friendly GUI 
Error checking
Game state management

------Technologies------
Java 
Java Swing 
Object-Oriented Programming (OOP)

------Skills Demonstrated----------
Algorithms 
Object-Oriented Programming 
Problem Solving 
GUI Development 
Java Software Development
